Data for Agents

It provides deep insights into NVIDIA's Nemotron open data strategy, highlighting how synthetic data is used to train robust agentic models and preserve proprietary workflows.
  • NVIDIA's Nemotron open data products (like Nemotron-CC and Nemotron-CC-MATH) leverage synthetic data to enhance Common Crawl and improve multi-step reasoning.
  • Synthetic data allows companies to train models on proprietary workflows ('secrets') without exposing the underlying sensitive data sources.
  • Building reliable agents requires training on software engineering traces, tool-use failures, and workflow execution data rather than just standard benchmarks.

Hugging Face Models on Foundry Managed Compute

It details a new managed GPU platform-as-a-service (Foundry Managed Compute) that abstracts GPU topology while supporting key inference runtimes like vLLM and SGLang.
  • Foundry Managed Compute acts as a managed GPU PaaS where developers specify parameter count and context length, and the platform handles the underlying GPU topology.
  • It natively supports and automatically patches major open-source inference runtimes including vLLM, SGLang, TensorRT-LLM, NIM, TEI, and llama.cpp.
  • The platform integrates with Foundry Agent Service for multi-agent orchestration, memory, and continuous evaluation loops.

Run AI workloads on any cloud, store on Hugging Face: zero-egress storage with SkyPilot

It introduces a zero-egress, cross-cloud storage solution using SkyPilot and Hugging Face, solving a major ML infra pain point of data gravity and transfer costs.
  • SkyPilot now supports Hugging Face Storage as a first-class backend (store: hf), allowing read-write mounting of buckets across 20+ clouds, Kubernetes, and Slurm.
  • Hugging Face Storage charges zero egress or CDN fees, eliminating the cross-cloud transfer tax when pulling datasets or models to available GPUs.
  • The storage backend is built on Xet, enabling block-level deduplication so incremental model checkpoints and variants only transfer changed chunks.

Hugging Face and Cerebras bring Gemma 4 to real-time voice AI

It showcases a low-latency, cascaded speech-to-speech pipeline using Cerebras hardware for fast LLM inference, though it is somewhat promotional for Cerebras.
  • The open, cascaded speech-to-speech stack consists of Nvidia's Parakeet (STT), Gemma 4 31B on Cerebras (VLM), and Qwen3TTS (TTS).
  • Cerebras hardware is leveraged to solve the language-model response time bottleneck, specifically targeting P95 latency spikes in multi-turn interactions.
  • This pipeline is already deployed in production to power over 9,000 Reachy Mini robots in the wild.

ScarfBench: Benchmarking AI Agents for Enterprise Java Framework Migration

While focused on Java framework migration, it introduces a novel, execution-based benchmark (ScarfBench) for evaluating how AI agents handle complex, real-world software engineering tasks.
  • ScarfBench evaluates AI agents on 204 migration tasks across Spring, Jakarta EE, and Quarkus, focusing on whether migrated apps actually build, deploy, and pass behavioral validation.
  • Unlike traditional code-generation benchmarks, it tests an agent's ability to handle dependency injection, build systems, and runtime dependencies.
  • The benchmark highlights that framework migration requires translating framework semantics and resolving complex application dependencies, not just translating source code.
