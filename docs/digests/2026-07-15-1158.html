Native-speed vLLM transformers modeling backend

Directly advances the reader's priorities in inference serving and performance engineering by optimizing the Hugging Face transformers backend for vLLM.

Run AI workloads on any cloud, store on Hugging Face: zero-egress storage with SkyPilot

Solves a major ML infrastructure pain point regarding data gravity and GPU availability by enabling zero-egress cross-cloud storage mounting.
  • Integrates Hugging Face Storage as a first-class SkyPilot backend (`store: hf`), allowing read-write mounting of HF Buckets via FUSE.
  • Eliminates egress fees when reading models and datasets from Hugging Face to GPUs located on any of 20+ supported clouds.
  • Leverages Xet-backed deduplication to ensure incremental checkpoints and model variants only transfer changed chunks.

🤗 Kernels: Major Updates

Highly relevant to performance engineering and GPU platforms, introducing a secure and reproducible way to distribute custom GPU kernels.
  • Introduces 'kernel' as a first-class repository type on Hugging Face to standardize custom GPU kernel distribution.
  • Uses Nix to achieve reproducible, hermetic builds of native GPU kernels, embedding the Git SHA for provenance.
  • Implements a security model with 'trusted kernel publishers' and code signing to prevent malicious code execution during Python loading.
