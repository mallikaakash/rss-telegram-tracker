Welcome Inkling by Thinking Machines

Deep technical breakdown of Inkling, a 1T parameter open multimodal MoE model with novel attention and inference features.
  • Inkling is a 975B total / 41B active parameter MoE model with 256 experts and a 1M context window.
  • Uses relative attention (instead of RoPE) to learn positions directly in attention logits, combined with a 5:1 hybrid sliding-window to global attention pattern.
  • Supports speculative Multi-Token Prediction (MTP) drafters and ships with native FP4 (NVFP4) quantization and day-0 vLLM/SGLang support.

Native-speed vLLM transformers modeling backend

Major performance engineering update enabling native-speed vLLM inference directly from Hugging Face transformers code.
  • The new transformers backend in vLLM meets or beats hand-written native implementations in throughput using the '--model-impl transformers' flag.
  • Eliminates the need for model authors to manually port architectures to vLLM-specific codebases to get optimized kernels.
  • Demonstrated parity/improvements on Qwen3 models, including a 235B FP8 MoE running on data and expert parallelism.

Dense Arena Interning: The Engine of Compiler Performance

Deep dive into compiler performance optimization via string interning, relevant to general systems and performance engineering.
  • Implements a Dense Arena Interner to convert strings and structures into dense integers upfront during lexing.
  • Replaces O(N*L) linear scans and O(L) hash map lookups with O(1) array indexing and single-instruction pointer comparisons.
  • Reduces memory overhead and cache misses by utilizing contiguous arena allocation.
