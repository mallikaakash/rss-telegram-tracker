Model Routing Is Simple. Until It Isn’t.

A brilliant systems-level deep-dive showing how context caching completely changes the economics of LLM routing.
  • Demonstrates that Claude Sonnet 4.6 was 50% cheaper than GPT-4.1 on AppWorld despite higher base pricing, due to Sonnet's lower cache-read costs.
  • Argues that task complexity is often invisible at routing time, making static classification-based routing ineffective.
  • Frames routing as a multi-dimensional systems optimization problem balancing cost, latency, caching, and compliance.
Must readHugging Face Blog

Welcome Inkling by Thinking Machines

A massive 1T parameter open MoE model release featuring highly relevant architectural innovations like relative attention and MTP.
  • Inkling is a 1T parameter open multimodal MoE model (41B active parameters, 256 experts) with a 1M context window.
  • Replaces RoPE with relative attention projections and uses a 5:1 hybrid of sliding window and global attention layers.
  • Includes native Multi-Token Prediction (MTP) drafters for speculative decoding, supported out-of-the-box in vLLM and SGLang.

Data for Agents

Directly addresses LLM training priorities by detailing NVIDIA's open and synthetic data strategies for training agents.
  • Argues that open weights are insufficient for agents; developers need open datasets, curation recipes, and training methods to ensure reproducibility.
  • Highlights NVIDIA's Nemotron open datasets (Nemotron-CC, Nemotron-CC-MATH) which use synthetic data to scale reasoning.
  • Explains how synthetic data allows enterprises to train models on proprietary workflows without exposing sensitive raw data.
