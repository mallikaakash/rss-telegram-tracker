Welcome Inkling by Thinking Machines

Highly technical architectural breakdown of the new 1T-parameter Inkling model, covering its unique attention mechanism and inference support.
  • Inkling is a decoder-only MoE with 975B total / 41B active parameters, 256 experts, and a 1M token context window.
  • Uses relative attention (injecting distance directly into attention logits) instead of RoPE, and a 5:1 hybrid ratio of sliding window to global attention.
  • Features speculative Multi-Token Prediction (MTP) layers and has day-0 support in vLLM, SGLang, and llama.cpp.

Technical deep dive into optimizing Inkling's hybrid attention on Modal using custom DFlash block-diffusion speculative decoding.
