Model Routing Is Simple. Until It Isn’t.

A brilliant, counter-intuitive engineering deep-dive on why simple model routing fails in production due to caching, task complexity, and latency.
  • Sticker price is misleading: Claude Sonnet 4.6 was 50% cheaper than GPT-4.1 on AppWorld because Sonnet's lower cache-read pricing heavily rewarded high context-cache hit rates.
  • Task complexity is invisible at routing time; seemingly simple prompts can trigger complex multi-step tool execution, making static routing ineffective.
  • Optimizing for latency requires predicting prompt/generation lengths and managing cold starts, turning routing into a multi-dimensional systems optimization problem.

Welcome Inkling by Thinking Machines

A major open-model release detailing a 1T parameter multimodal MoE architecture with novel attention mechanisms and inference optimizations.
  • Inkling is a 1T parameter (41B active) multimodal MoE model supporting text, image, and audio inputs with a 1M context window.
  • Replaces RoPE with relative attention (learning position directly in attention logits) and uses a 5:1 hybrid ratio of sliding window to global attention layers.
  • Features speculative Multi-Token Prediction (MTP) drafters for faster inference, with day-0 support in SGLang, vLLM, and llama.cpp.
