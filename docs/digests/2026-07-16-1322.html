Welcome Inkling by Thinking Machines

A major open model release featuring a 1T MoE architecture, novel relative attention mechanisms, and speculative multi-token prediction.
  • Inkling is a 1T parameter multimodal MoE model (41B active, 256 experts) with a 1M context window.
  • Replaces RoPE with a relative attention mechanism that learns position directly in attention logits via a fourth projection tensor.
  • Uses hybrid attention (5:1 sliding window to global) and speculative Multi-Token Prediction (MTP) drafters for optimized inference.
  • Features day-0 integration with SGLang, vLLM, and llama.cpp.
