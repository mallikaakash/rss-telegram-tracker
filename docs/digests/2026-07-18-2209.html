Fixing Gradient Accumulation

This is a critical technical deep-dive into a mathematical discrepancy in Hugging Face's gradient accumulation, directly impacting LLM training and loss convergence.
  • Identifies a bug where gradient accumulation was not mathematically equivalent to full batch training due to how default token-level loss was averaged.
  • Explains that the correct loss must divide the total loss across all accumulated batches by the total number of non-padding tokens, rather than averaging per-batch losses.
  • Introduces a fix in the Hugging Face `transformers` Trainer to automatically handle this scaling when using gradient accumulation.

Improving Parquet Dedupe on Hugging Face Hub

A high-quality engineering deep-dive into optimizing Parquet file storage and deduplication on the Hugging Face Hub, directly relevant to ML data infrastructure.
  • Analyzes why standard Content-Defined Chunking (CDC) struggles with Parquet files due to absolute file offsets in column headers rewriting on small modifications.
  • Shows that appending rows dedupes extremely well (99.1% deduped), whereas modifying a single row causes widespread header changes, dropping deduplication to 89%.
  • Highlights Hugging Face's ongoing efforts to optimize its 2.2PB Parquet storage footprint through layout-aware deduplication.
