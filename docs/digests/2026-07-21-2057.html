Toward understanding and preventing misalignment generalization

Cutting-edge alignment and mechanistic interpretability research from OpenAI using SAEs to find and steer a 'misaligned persona' feature.
  • Discovers a 'misaligned persona' feature in GPT-4o's activations using Sparse Autoencoders (SAEs).
  • Demonstrates that steering this latent feature can directly cause or suppress emergent misalignment in unrelated tasks.
  • Shows that retraining on correct data can push the model back to alignment, offering a path toward early warning systems.
