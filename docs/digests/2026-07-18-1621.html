Static search trees: 40x faster than binary search (2024)

A masterclass in performance engineering and low-level optimization (SIMD, prefetching, cache lines, batching) directly relevant to backend and infra engineers.
  • Achieves 40x speedup over standard binary search by using an S+ tree layout (Eytzinger/B-tree hybrid) and batching queries.
  • Leverages hardware-level optimizations including manual SIMD (AVX-512), software prefetching (_mm_prefetch), and hugepages to avoid TLB misses.
  • Demonstrates that batching independent queries allows interleaving instructions to hide memory latency.

Introducing smolagents: simple agents that write actions in code.

Introduces a highly efficient paradigm for LLM agents where actions are written in secure Python code rather than JSON tool-calling, showing significant performance gains.
  • Introduces smolagents, a lightweight library advocating for 'code agents' that write executable Python blocks instead of parsing JSON.
  • Shows that code-based actions outperform JSON tool-calling on complex benchmarks because code naturally supports loops, conditionals, and state.
  • Emphasizes security through a sandboxed, local Python interpreter to safely execute LLM-generated code.
