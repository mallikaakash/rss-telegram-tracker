Welcome Inkling by Thinking Machines

This is a major open 1T-parameter multimodal MoE model release featuring novel architectural choices like relative attention and hybrid global/sliding-window attention.
  • Inkling is a 975B parameter (41B active) sparse MoE model with 256 experts, supporting a 1M context window across text, image, and audio.
  • Replaces RoPE with relative attention (learning position directly in attention logits) and alternates global attention with sliding window attention (5:1 ratio).
  • Supports speculative Multi-Token Prediction (MTP) drafters for faster inference and comes in a well-calibrated NVFP4 quantized variant.

What building Shippy taught us about building agents

It provides practical architectural patterns for building reliable, sandboxed, and verifiable AI agents in high-stakes domains.
  • Agents are structured into three components: 'soul' (system prompt), 'skills' (versioned markdown files with structured frontmatter), and 'config' (runtime settings and model selection).
  • The system uses OpenClaw as the agent framework, Claude Opus 4.6 as the LLM, and sandboxed Docker containers for execution.
  • Emphasizes evaluating the entire agent system against live, continuously updated data rather than evaluating the raw LLM on static benchmarks.
