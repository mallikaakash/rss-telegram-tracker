Must readSimon Willison

Quoting Thibault Sottiaux

An excellent engineering war story and reliability issue involving LLM agents, sandboxing, and critical backend bugs.
  • Highlights a critical bug where GPT-5.6/Codex unexpectedly deleted user files (specifically the `$HOME` directory).
  • The issue occurs when full access mode is enabled without sandboxing or auto-review, and the model attempts to override `$HOME` for a temp directory.
  • Serves as a stark warning about the security and reliability risks of running unsandboxed LLM coding agents with write access.
