SQLite Is All You Need

A highly practical performance engineering deep-dive showing how SQLite in WAL mode can handle 315M requests/day on a single laptop.
  • Demonstrates that a single-process Node API backed by SQLite in WAL mode can serve 3,543 req/s for a heavy timeline join (with 2.5M follows and 1M posts) on an M1 laptop.
  • Key configuration pragmas used: journal_mode = WAL (non-blocking reads/writes), synchronous = NORMAL (fsync at checkpoints), and cache_size = -64000 (64MB page cache).
  • Highlights that SQLite is highly viable for production workloads, challenging the default choice of spinning up a Postgres container.

Google Cloud C4 Brings a 70% TCO improvement on GPT OSS with Intel and Hugging Face

Directly aligns with Priority 1 (MoE architectures) and Priority 2 (ML infrastructure, performance engineering, CPU inference).

  • Highlights a key software optimization (PR #40304 in Hugging Face transformers) that routes tokens only to active experts, eliminating redundant FLOPs.
