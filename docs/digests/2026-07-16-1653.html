Welcome Inkling by Thinking Machines

Directly targets core priorities: a massive 1T open MoE model featuring relative attention, hybrid attention, and speculative decoding.
  • Inkling is a 1T parameter (41B active) decoder-only multimodal MoE model with 256 experts and a 1M context window.
  • Replaces RoPE with relative attention (learning position directly in attention logits) and uses a 5:1 ratio of sliding window to global attention layers.
  • Features speculative Multi-Token Prediction (MTP) drafters for faster inference and ships with day-0 support in SGLang, vLLM, and NVFP4 quantization.
