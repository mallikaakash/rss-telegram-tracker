Welcome Inkling by Thinking Machines

Introduces a massive 1T parameter open multimodal MoE model with novel architectural details like relative attention and hybrid attention.
  • Inkling is a 975B parameter MoE model (41B active, 256 experts) supporting text, image, and audio inputs with a 1M context window.
  • Replaces RoPE with relative attention (learned directly in attention logits) and uses a 5:1 hybrid ratio of sliding window to global attention.
  • Features speculative Multi-Token Prediction (MTP) drafters and has day-0 support in SGLang, vLLM, and llama.cpp.
